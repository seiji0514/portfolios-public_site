現場スタッフ向け 脳トレゲーム操作説明

1. 利用者への案内方法

  1. 院内WiFiに接続されていることを確認してください。
  2. 利用者の端末（PC/タブレット/スマホ）でブラウザを開きます。
  3. アドレスバーに http://[サーバーPCのIPアドレス]/version2/index_limited.html
  4. ログイン画面が表示されたら、利用者にユーザー名・パスワードを入力してもらいます。
  5. 「ログイン」ボタンを押すと、ゲーム選択画面に進みます。

2. ゲームの始め方

  1. 利用者が好きなゲームを選択します。
  2. 「開始」ボタンを押すとゲームが始まります。
  3. ゲーム終了後、スコアや結果が表示されます。

3. よくある質問・トラブル対応

  • 画面が表示されない： WiFi接続やアドレスの入力ミスを確認してください。
  • ログインできない： ユーザー名・パスワードを再確認し、管理者に連絡してください。
  • ゲームが動かない： ブラウザを更新、または端末を再起動してください。

4. サポート・連絡先

  • 不明点やトラブルがあれば、管理者またはシステム担当までご連絡ください。
