院内 WiFi 接続設定ガイド

概要

院内 WiFi 環境でのシステム接続設定について説明します。

接続先設定

現在の設定（要変更）

  • サーバー IP: 192.168.1.100 ← 実際の院内 IP に変更が必要
  • API ポート: 3000
  • フロントエンドポート: 5500

院内 WiFi での接続先

APIサーバー: http://[院内IP]:3000
フロントエンド: http://[院内IP]:5500

設定変更手順

1. IP アドレスの確認

# Windowsの場合
ipconfig

# 院内WiFiのIPアドレスを確認
# 例: 192.168.10.50

2. 設定ファイルの変更

config.js

server: {
    localIP: '192.168.10.50', // 実際の院内IPに変更
    publicIP: '192.168.10.50'  // 実際の院内IPに変更
}

port_config.js

LOCAL_IP: '192.168.10.50', // 実際の院内IPに変更
API_BASE_URL: 'http://192.168.10.50:3000',
FRONTEND_BASE_URL: 'http://192.168.10.50:5500'

3. サーバー起動

node server.js

接続確認

ブラウザでの確認

  1. http://[院内IP]:3000 - API サーバー
  2. http://[院内IP]:5500 - フロントエンド

コンソールでの確認

// ブラウザのコンソールで実行
window.checkHospitalWiFiConnection();

注意事項

  • 院内 WiFi の IP アドレスは固定 IP を推奨
  • ファイアウォールでポート 3000 と 5500 を開放
  • セキュリティ設定の確認が必要
