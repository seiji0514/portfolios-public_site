脳トレゲーム 操作マニュアル

1. システム起動方法

サーバー起動手順（管理者向け）

  1. デスクトップの「QRコード認証」フォルダを開く
  2. version2フォルダに移動
  3. api_server.jsを実行
    コマンドプロンプトで node api_server.js を実行。「API server running at http://localhost:3000」と表示されることを確認

アクセス方法（利用者向け）

  1. 院内WiFiに接続
  2. ブラウザ（Chrome推奨）を起動
  3. アドレスバーに以下を入力：
    http://192.168.1.6:5174/plateau_viewer.html
  4. PIN認証登録画面が表示されることを確認
  5. 初回利用時はPIN認証登録が必要です

※ サーバーPC上で直接テストする場合は、アドレスに http://localhost:5174/plateau_viewer.html を使用することも可能です。

2. 基本操作

ログイン方法（PIN認証システム）

  1. PIN認証登録ページにアクセス
  2. 個人情報（姓名、生年月日、性別、連絡先）を入力
  3. 職種（管理者/スタッフ）を選択
  4. 6桁以上の英数字混合PINを設定
  5. PINの確認入力を行う
  6. 「登録完了」ボタンをクリック
  7. 登録完了後、メインシステムに自動遷移

※ PIN認証は30分間有効です。30分経過後は再認証が必要になります。

ゲーム選択・プレイ方法

  1. メイン画面でゲームを選択
  2. 各ゲームの説明を確認
  3. 「開始」ボタンでゲーム開始
  4. ゲーム終了後、結果が表示される

眼の訓練メニュー

  1. 「眼の訓練」カードを選択
  2. 訓練メニューから希望のものを選択
    • 追視訓練
    • 周辺視野訓練
    • 動体視力訓練
    • 反応速度訓練

3. 管理者機能

管理画面へのアクセス

  1. PIN認証登録時に職種で「管理者」を選択
  2. メインシステムにアクセス後、「利用履歴」メニューを選択
  3. 管理者権限で利用履歴の確認が可能

ゲーム管理

  1. ゲームの追加・編集・削除
  2. 表示/非表示の設定
  3. 難易度調整

お知らせ管理

  1. お知らせの追加
  2. 既存のお知らせの編集・削除

進捗確認

  1. ユーザーごとの進捗確認
  2. スコア履歴の確認

4. トラブルシューティング

接続できない場合

  • WiFi接続を確認
  • サーバーの起動状態を確認
  • IPアドレスの確認

ゲームが動作しない場合

  • ブラウザの更新
  • キャッシュのクリア
  • 別のブラウザでの試行

PIN認証でログインできない場合

  • PINの入力ミスを確認
  • アカウントロック状態を確認（5回失敗で30分ロック）
  • セッション期限切れの場合は再ログイン

5. 注意事項

  • サーバーPCの電源を切る前に、必ずサーバーを停止する
  • 定期的なバックアップを推奨
  • 個人情報の取り扱いに注意
  • PINデータはローカルストレージに安全に保存されます
  • PINは6桁以上の英数字混合で設定してください
  • PIN認証は30分間有効です。期限切れの場合は再認証が必要です
  • 5回のログイン失敗でアカウントが30分間ロックされます
