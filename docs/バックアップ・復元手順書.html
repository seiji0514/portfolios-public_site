バックアップ・復元手順書

1. バックアップの取得方法

  1. 管理画面にログインします。
  2. 「バックアップ管理」メニューを選択します。
  3. 「バックアップ作成」ボタンをクリックします。
  4. バックアップファイルが自動で保存されます（保存先：version2/backup_日付/）。

2. 復元方法

  1. 管理画面の「バックアップ管理」を開きます。
  2. 復元したいバックアップの「復元」ボタンをクリックします。
  3. 確認メッセージが表示されたら「OK」を押します。
  4. 復元が完了したら、システムを再起動してください。

3. 注意事項

  • 定期的にバックアップを取得してください。
  • 重要なデータは外部メディアにも保存しておくと安心です。
  • 復元操作は管理者のみが行ってください。

バックアップ・復元に不安がある場合は、システム担当者にご相談ください。
