## 1. 法令遵守の基本ポイント

### 1.1 個人情報の取り扱い
- 患者さんの情報は絶対に漏らさない
- 必要な人だけがアクセスできるようにする
- 情報の開示や訂正の依頼があったらすぐに対応

### 1.2 医療情報システムの運用
- 厚生労働省のルールに従う
- 情報漏洩が起きた場合の対応手順を決めておく
- 定期的にセキュリティチェックを行う

### 1.3 セキュリティ対策
- PIN認証による本人確認システムを導入
- 顔認証からPIN認証方式に変更し、セキュリティを強化
- 不正アクセスを防ぐ
- 問題が起きた時の対応手順を決めておく

### 1.3.1 PIN認証システムのセキュリティ機能
- システムアクセス前のPIN認証必須化
- 6桁以上の英数字混合PINによる認証
- 30分有効なセッション管理
- 5回失敗で30分間アカウントロック
- 利用状況の自動記録と監視
- 管理者による利用履歴確認機能

### 1.4 アクセシビリティ対応
- 高齢者や障害のある方も使いやすい設計
- 画面の見やすさ、操作のしやすさに配慮
- 必要な支援を提供できる体制

### 1.5 著作権の遵守
- 使用するソフトウェアのライセンスを確認
- コンテンツの利用権限を確認
- オープンソースのルールを守る

### 1.6 労働管理
- 適切な労働時間の管理
- 休憩時間の確保
- 安全な作業環境の維持

## 6. システムアクセス情報

### 6.1 アクセス権限区分
1. 管理者（PIN認証登録時に「管理者」を選択）
2. スタッフ（PIN認証登録時に「スタッフ」を選択）

### 6.2 PIN認証システム情報

#### 6.2.1 PIN認証登録手順
1. システム初回アクセス時にPIN認証登録ページに自動リダイレクト
2. 個人情報（姓名、生年月日、性別、連絡先）を入力
3. 職種（管理者/スタッフ）を選択
4. 6桁以上の英数字混合PINを設定
5. PINの確認入力を行う
6. 登録完了後、メインシステムにアクセス可能

#### 6.2.2 セッション管理
- 有効期限：30分
- 再認証：30分経過後に自動的に必要
- ログアウト：手動ログアウトボタンで即座に無効化
- アカウントロック：5回失敗で30分間ロック

#### 6.2.3 利用履歴管理
- 自動記録：ゲーム開始・終了、利用時間、スコア等
- 管理者確認：利用履歴ページで全利用者の状況を確認可能
- データ保持：ローカルストレージに保存

### 6.3 セキュリティ注意事項
1. PINデータの安全な管理
2. 個人情報の適切な取り扱い
3. アクセス権限の適切な管理
4. 不審なアクセスの監視
5. 利用履歴の定期的な確認

### 6.4 PIN認証システム運用ルール

#### 6.4.1 登録時の注意事項
1. 正確な個人情報の入力
2. 6桁以上の英数字混合PINの設定
3. 連続数字や簡単なPINの使用禁止
4. 登録完了の確認

#### 6.4.2 日常利用時の注意事項
1. セッション有効期限の確認
2. 利用終了時のログアウト
3. 共有端末での利用時の注意
4. PINの安全な管理

#### 6.4.3 セキュリティ強化事項
1. アカウントロック機能の活用
2. 定期的なPIN変更の推奨
3. 不審なアクセスの監視
4. 利用履歴の定期的な確認
